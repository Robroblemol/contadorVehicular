1.-Empesando desde un repositoriio local

Para subir loa cambio a un repositorio remoto

a. usuario en git.com confiigurado
b. crear proyecto en github
c. vinculamos el repositorio con el local

git remote add nombreX dirrecion de repositorio terminado en .git
git remote -v // para ver lo que tenemos en el repositorio remoto

d. Empujamos los cambios ya dados de alta (git add -A , git commit -m "comentario")
git push nombreX master

2. creando primero el repositorio en github y lugo descargarlo en el equipo

a.creamos el proyecto en github
b.en la carpeta que qeremos que se guarde insertamos comando
git clone <direccion repositorio remoto.git>
git add <nombre del archivo a guardar>
git commit -m "comentario"
git push origin master

>>>>>>>>>>>>>>>>>>>>Comando interesante de git<<<<<<<<<<<<<<<<<<<<<<<<<<<<<
git status // para ver si hay cambios
git add -A // para agregar todos los archivos en el directorio
git add <nombre del archivo> // para agregarr solo el archivo especifico
git remote -v // para ver lo que tenemos en el repositorio remoto
git rm --cached <nombre del archivo> para eliminar archivos del repositorio>
git remote rename <Nombre del identificador> <Nuevo nombre del identifidador>
git --help // ver los comandos disponibles en git
git log // para ver los cambio realizados
git log --oneline//para ver los cambios realizado resumido
git add .// para subir todos los archivos modificados a repositorio
git push <identificador> rama
